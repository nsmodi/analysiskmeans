# Configure and Normalize input data

Configure and Normalize input data

## Usage

``` r
data_config(sce)
```

## Arguments

- sce:

  A SingleCellExperiment object

## Value

List of A SingleCellExperiment subset, List of cell type labels and Log
Normalized SingleCellExperiment object

## Examples
