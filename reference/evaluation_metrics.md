# Compute ARI Values for each K

Compute ARI Values for each K

## Usage

``` r
evaluation_metrics(min_k, max_k, km_list, cell_type)
```

## Arguments

- max_k:

  The max amount k value you wish to go up to

- km_list:

  List of K Means clustering results with various K values

- cell_type:

  Corresponding cell type labels

## Value

ARI values for each K
